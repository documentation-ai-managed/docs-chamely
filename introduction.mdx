## Overview

Chamely provides a centralized space for you to create, organize, and collaborate on project documentation. You streamline workflows by keeping all docs in one intuitive platform, reducing scattered files and version confusion. Access everything from markdown files to interactive guides in a single, searchable hub.

<Callout kind="info">
  Chamely supports teams of all sizes, from solo developers to enterprise groups.
</Callout>

## Quick Start

Follow these steps to set up your first documentation space:

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create an account with your email.
  </Step>
  <Step title="Create a Space" icon="folder-plus">
    Click "New Space" and name it for your project, like "MyApp Docs".
  </Step>
  <Step title="Add Your First Doc" icon="file-plus">
    Create a new page and start writing in markdown.

````markdown
# Welcome to MyApp

This is your project hub.

- [Quickstart](/quickstart)
- [API Reference](/authentication)
````

  </Step>
  <Step title="Invite Collaborators" icon="mail">
    Share the space link or add emails via the settings panel.
  </Step>
</Steps>
