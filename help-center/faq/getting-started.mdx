## Overview

Chamely makes it easy to start documenting your project.

## Quick Steps

1. Log in to your Chamely dashboard
2. Click "New Project" in the top right
3. Enter a name and description
4. Click "Create" to start organizing your docs

You can now add pages, guides, or import existing docs to your Chamely workspace.
