## Get Started

<Steps>
  <Step title="Go to your Chamely dashboard">
    ````info
    Log in at https://app.chamely.com
    ````
  </Step>
  <Step title="Click 'New Project'">
    ````tip
    Find the button in the upper right corner.
    ````
  </Step>
  <Step title="Fill out project details">
    ````info
    Name your project and add a description or tags.
    ````
  </Step>
  <Step title="Finalize">
    ````tip
    Click "Create". You can now add or edit pages!
    ````
  </Step>
</Steps>
