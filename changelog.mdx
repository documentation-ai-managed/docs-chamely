## How to Upgrade

Upgrade Chamely to the latest version using your preferred package manager. Always back up your documentation site first.

<Steps>
  <Step title="Backup Your Site" icon="database">

    Export your current documentation:

````bash
chamely export --output ./backup
````

  </Step>
  <Step title="Update Package" icon="package">
    <Tabs>
      <Tab title="npm" icon="nodejs">
````bash
npm update chamely
````
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn upgrade chamely
````
      </Tab>
      <Tab title="pnpm" icon="archive">
````bash
pnpm update chamely
````
      </Tab>
    </Tabs>
  </Step>
  <Step title="Restart Server" icon="refresh-cw">

    Restart your Chamely instance:

````bash
chamely start --port 3000
````

  </Step>
</Steps>

<Callout kind="tip">
  Check the [release notes](https://github.com/chamely/releases) for migration guides if you encounter breaking changes.
</Callout>
